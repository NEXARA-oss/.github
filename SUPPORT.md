# Getting Support

## Where to Ask

| Channel | Best For | Response Time |
|---|---|---|
| [Discord #help](https://discord.gg/nexara) | Quick debugging, general questions | Hours |
| [GitHub Discussions](https://github.com/nexara-org/neuroweave/discussions) | Design questions, technical deep-dives | 1–3 days |
| [Stack Overflow](https://stackoverflow.com/questions/tagged/nexara) | Searchable Q&A for common problems | Community |
| [security@nexara.dev](mailto:security@nexara.dev) | Security vulnerabilities **only** | 48 hours |

## Before Asking

1. Read the [documentation](https://docs.nexara.dev)
2. Check the [FAQ](https://docs.nexara.dev/faq)
3. Search existing [issues](https://github.com/nexara-org/neuroweave/issues) and [Discussions](https://github.com/nexara-org/neuroweave/discussions)
4. Confirm you're on the latest version

## Writing a Good Help Request

Include:
- Your `@nexara/core` version (`pnpm list @nexara/core`)
- Node.js version and OS
- A minimal code snippet reproducing the issue
- What you expected vs. what actually happened
- Full error output — not just the last line

The more context you provide, the faster you get help.
